# NMsolver

ニュートン法を用いて非線形方程式を解くソルバーです。
このリポジトリには Julia 版と Fortran 版の実装があります。

## 共通点

- 1変数方程式 `f(x) = 0` をニュートン法で解く
- 多変数連立方程式 `F(x) = 0` をニュートン法で解く
- 収束判定には関数値や更新量の大きさを使う
- テスト用の非線形方程式を用意し，計算した解と真の解を比較する

## Julia 版

Julia 版は `julia/` にあります。

- ソルバー本体は `julia/src/NMsolver.jl`
- テストは `julia/test/runtests.jl`
- 性能評価は `julia/scripts/evaluate.jl`
- `Project.toml` により Julia のプロジェクト環境として管理
- 導関数やヤコビアンを省略した場合，中心差分による数値微分で自動計算する
- 1〜10変数の問題について，誤差・反復回数・計算時間を評価する
- 評価結果をグラフとして `julia/results/` に保存する

Julia 版は，ソルバーの使いやすさと性能評価を重視した実装です。

## Fortran 版

Fortran 版は `Fortran/` にあります。

- ソルバー本体は `Fortran/newton_solver.f90`
- テストプログラムは `Fortran/test_newton.f90`
- `gfortran` でコンパイルして実行する
- 導関数やヤコビアンは利用者が明示的に定義して渡す
- 多変数ニュートン法の内部で必要な線形方程式を，自前実装の LU 分解で解く
- 1変数，2変数，3変数の例題で動作確認する

Fortran 版は，ニュートン法の各ステップや線形方程式の解法を明示的に確認しやすい実装です。

## Julia 版と Fortran 版の主な違い

| 項目 | Julia 版 | Fortran 版 |
|---|---|---|
| 実装場所 | `julia/` | `Fortran/` |
| ソルバー本体 | `NMsolver.jl` | `newton_solver.f90` |
| 導関数の扱い | 省略時は数値微分で近似 | 解析的導関数を明示的に渡す |
| ヤコビアンの扱い | 省略時は数値ヤコビアンを自動計算 | 解析的ヤコビアンを明示的に渡す |
| 線形方程式の解法 | Julia 標準の `\` を使用 | 自前実装の LU 分解を使用 |
| 評価機能 | 誤差・反復回数・計算時間を評価しグラフ化 | 主に例題による動作確認 |
| 対象次元 | 評価では 1〜10 変数 | テストでは 1〜3 変数 |
| 実行環境 | Julia の project 環境 | `gfortran` によるコンパイル実行 |

## 使い分け

- Julia 版は，数値微分や性能評価まで含めて手軽に試したい場合に向いています。
- Fortran 版は，導関数・ヤコビアン・LU 分解を明示的に扱い，アルゴリズムの中身を確認したい場合に向いています。
